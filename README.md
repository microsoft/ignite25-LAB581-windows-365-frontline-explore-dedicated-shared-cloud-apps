# [Microsoft Ignite 2025](https://ignite.microsoft.com)

## 🔥LAB581: Windows 365 Frontline: Explore Dedicated, Shared, Cloud Apps

[![Microsoft Azure AI Foundry Discord](https://dcbadge.limes.pink/api/server/ByRwuEEgH4)](https://aka.ms/AIFoundryDiscord-Ignite25)
[![Azure AI Foundry Developer Forum](https://img.shields.io/badge/GitHub-Azure_AI_Foundry_Developer_Forum-blue?style=for-the-badge&logo=github&color=adff2f&logoColor=fff)](https://aka.ms/AIFoundryForum-Ignite25)

### Session Description

*Go from zero to hero in this hands-on lab exploring Windows 365 Frontline. Learn to create and configure Dedicated, Shared, and Cloud Apps environments. Understand when to use each model for shift workers, task-based roles, or app-only access. Walk through provisioning, Entra ID, Intune, and Conditional Access. Experience the user journey and leave ready to optimize for performance, cost, and compliance.*


### 💻 Technologies Used

1. Windows 365 Frontline
1. Intune
1. Windows App
